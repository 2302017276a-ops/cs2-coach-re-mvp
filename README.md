# Donk 又红温了？

![Version](https://img.shields.io/badge/version-v0.2.0-65f0a2)

当前版本：`v0.2.0`

一个纯前端的 CS2 电竞战队文字模拟器 MVP。

## 怎么运行

直接双击 `index.html` 即可体验。

也可以用任意静态服务器运行，例如 VS Code 的 Live Server。

## 当前功能

- 一个赛季 5 周
- 每周选择一个行动
- 事件链驱动叙事，而不是单次随机事件堆量
- 重点事件链：那个总决赛、那个叛徒、那个天才新人、那次翻盘、那次崩盘
- 战队属性：资金、士气、名气、战术熟练度、团队默契、粉丝支持
- 选手属性：枪法、意识、心态、团队性、商业价值、疲劳
- 选手人格：自尊、忠诚、残局、红温倾向
- 比赛文字直播，先到 13 分的一方获胜
- 比赛关键节点由玩家决策
- 赛季结束后生成结局、评价和纪录
- 使用 Zustand vanilla 管理游戏状态
- 使用 LocalStorage 自动存档，刷新网页后继续游戏
- 无后端，可直接部署到 GitHub Pages

## 核心循环

```text
选择本周行动
推进一条事件链
更新队伍属性和选手人格
进行先到 13 分的文字直播比赛
关键节点教练决策
记录赛季名场面
进入下一周
```

