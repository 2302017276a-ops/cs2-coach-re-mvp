# Donk 又红温了？

![Version](https://img.shields.io/badge/version-v0.4.0-65f0a2)

当前版本：`v0.4.0`

一个纯前端的 CS2 电竞战队文字模拟器 MVP。

## 怎么运行

直接双击 `index.html` 即可体验。

也可以用任意静态服务器运行，例如 VS Code 的 Live Server。

## 当前功能

- 一个赛季 5 周
- 开局有限预算选购 5 名选手建队
- 选人池扩展到 40 名真实 CS2 选手，价格由五维评分推算
- 每周从随机行动池选择一个行动，不再固定
- 事件链和比赛局势共同驱动叙事，而不是单次随机事件堆量
- 重点事件链：那个总决赛、那个叛徒、那个天才新人、那次翻盘、那次崩盘
- 战队属性：资金、士气、名气、战术熟练度、团队默契、粉丝支持
- 选手属性：枪法、意识、心态、团队性、商业价值、疲劳
- 选手人格：残局能力、红温倾向、明星气质
- 引入对手队员数值和战术风格
- 比赛文字直播，点击推进每个回合，先到 13 分的一方获胜
- 每回合显示参考胜率，精确到小数点后两位
- 局势节点：经济局、关键残局、暂停、士气变化、赌点、信息误判、明星对位
- 玩家干预点：暂停、变速、强起、赌点、针对明星、安抚红温选手
- 右侧主动暂停按钮，单场 3 次
- 对手会根据你的打法作出调整
- 赛季结束后可选择继续磨合或变阵重组
- 赛后舆论生成器：论坛标题、比赛争议、选手评分
- 赛季结束后生成结局、评价和纪录
- 使用 Zustand vanilla 管理游戏状态
- v0.3.1 起已移除外部 CDN 依赖，避免选手市场加载失败
- 使用 LocalStorage 自动存档，刷新网页后继续游戏
- 无后端，可直接部署到 GitHub Pages

## 核心循环

```text
选择本周行动
随机抽取本周可选行动
选择本周比赛风格
进行先到 13 分的文字直播比赛
点击推进每个回合
关键节点教练干预
生成赛后舆论
记录赛季名场面
进入下一周
```
