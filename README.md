# CS2 Coach RE MVP

一个纯前端的 CS2 电竞教练文字冒险 MVP。

## 怎么运行

直接双击 `index.html` 即可体验。

也可以用任意静态服务器运行，例如 VS Code 的 Live Server。

## 当前功能

- 三个开局事件
- 战队八维属性
- 12 个比赛镜头跑完一场 MR12 风格短局
- 两次教练决策
- 20 个随机电竞圈事件
- 赛后截图卡
- 无后端、无依赖、可直接部署到 GitHub Pages

## 后续方向

- 增加长期养成和本地存档
- 加入战术、地图池、转会市场
